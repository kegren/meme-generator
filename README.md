# A small meme generator project

### Built using
* Silex
* Intervention Image
* Flysystem
* Angular
* Bower
* Foundation

### Requirements
* 5.4+
* GD
* Browser with support for FormData
* Supports jpg/jpeg,gif and png
* Works best with images with higher res

### Install
* Clone this repository
* Composer install
* php -S localhost:3000 (if local)
* Enjoy
